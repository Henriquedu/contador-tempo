# Quanto Falta?

Uma aplicação web simples e intuitiva para acompanhar uma contagem regressiva até qualquer data e horário.

Escolha um momento no futuro e veja, em tempo real, exatamente quanto falta para ele acontecer, com atualização automática a cada segundo.

## Demonstração

Você pode acessar o projeto pelo GitHub Pages:

**https://henriquedu.github.io/quanto-falta/**

> *(Atualize o link após publicar o repositório.)*

---

## Funcionalidades

- Escolha de qualquer data e horário futuro
- Contagem regressiva em tempo real
- Exibição de:
  - Dias
  - Horas
  - Minutos
  - Segundos
- Horas, minutos e segundos totais restantes
- Armazenamento da data utilizando LocalStorage
- Interface responsiva para desktop, tablet e smartphone
- Mensagens dinâmicas conforme a aproximação da data

## Como utilizar

1. Clone este repositório.

```bash
git clone https://github.com/Henriquedu/quanto-falta.git
```

2. Abra o arquivo `index.html` no navegador.

3. Escolha uma data e horário futuros.

4. Clique em **Salvar data**.

Pronto! A contagem regressiva começará automaticamente e permanecerá salva mesmo após fechar a página.
